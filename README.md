# 🧩 My Jigsaw Puzzle

好きな写真をアップロードして、自分だけのジグソーパズルを作って遊べるブラウザゲームです。
ライブラリ不使用・HTML / CSS / JavaScript だけで動きます。

## 遊び方

1. `index.html` をブラウザで開く(ダブルクリックで OK)
2. 好きな写真をドラッグ&ドロップ(またはクリックして選択)
   - 写真がなくても「🎨 サンプル画像で遊ぶ」ですぐ遊べます
3. ピースの数(難易度)を選んで「スタート!」
4. ばらまかれたピースをドラッグして、中央の枠にはめていこう
   - 正しい位置の近くで離すと **カチッ** とはまります
5. 全ピースをはめると完成!タイムと移動回数が表示されます

## 機能

- 📷 **好きな写真でパズル生成** — ドラッグ&ドロップ / ファイル選択に対応(EXIF の回転も自動補正)
- 🎚 **難易度 5 段階** — 12 / 24 / 54 / 96 / 192 ピース(写真の縦横比に合わせてピース形状を自動調整)
- 🧩 **本格的なピース形状** — ベジェ曲線による凹凸付きピースをランダム生成
- 💡 **ヒント機能** — 盤面にうっすら完成図を表示
- 🖼 **完成図の確認** — いつでも元の写真をチェック
- 🔀 **シャッフル** — ピースをばらまき直し
- ⏱ **タイム & 進捗表示** — 経過時間とはめたピース数をカウント
- 🔊 **効果音** — ピースがはまる音・完成ファンファーレ(WebAudio で生成)
- 🌊 **波の音** — 打ち寄せる波の環境音をリアルタイム生成(オン/オフ切替可)
- 🌅 **時間帯で変わる海** — 実際の時刻に合わせて昼・夕暮れ・星降る夜へと風景が変化(`?time=day|sunset|night` で固定も可能)
- ✨ **スナップエフェクト** — ピースがはまった瞬間に光の波紋がひろがる演出
- 📱 **タッチ対応** — スマホ・タブレットでも遊べます
- 📲 **PWA 対応** — ホーム画面に追加してアプリのように起動でき、2回目以降は**オフラインでも遊べます**(Service Worker によるキャッシュ)

## 動作環境

Chrome / Edge / Safari / Firefox などのモダンブラウザ。
サーバー不要でローカルファイルのまま動作します。

## ファイル構成

| ファイル | 内容 |
| --- | --- |
| `index.html` | 画面構成(セットアップ / ゲーム / クリア画面) |
| `style.css` | スタイル |
| `puzzle.js` | パズル生成・描画・ゲームロジック |
| `manifest.json` | PWA マニフェスト(アプリ名・アイコン・表示モード) |
| `sw.js` | Service Worker(オフラインキャッシュ) |
| `icons/` | PWA・ホーム画面用アイコン |

※ オフライン対応(Service Worker)は HTTPS または localhost で開いたときのみ有効です。`index.html` を直接開いた場合も、ゲーム自体は問題なく遊べます。
